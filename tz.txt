Сайт должен содержать 12 страниц и систему добавления контента:
/////--Main page - главная страница сайта; (https://mirelia.co/ )
/////--About us - страница описание нетворка (https://mirelia.co/about )
/////--Advertisers - страница посвящена рекламодателям (https://mirelia.co/advertisers )
/////--Affiliates - страница посвящена аффилиатам (https://mirelia.co/affiliate )
/////--Contact us - форма; свяжитесь с нами (https://mirelia.co/contact )
/////--Referral Program - реферальная программа сотрудничества с партнерами (https://mirelia.co/referral )
Affiliate Sign up - форма; заполняет аффилиаты для регистрации (https://mirelia.co/affiliate/signup )
/////--Privacy Policy - политика конфиденциальности ( https://mirelia.co/policy )
/////--Terms & Conditions - термзы (https://mirelia.co/terms )
/////--Tracking Policy - новая вкладка термзов; (https://mirelia.co/tracking )
/////--Blog - страница блога с поисковой строкой и статьями
/////--Blog article - статья блога

Технологии:
Предпочтительные технологии: html + css + native js для верстки фронта
Адаптивность:
Сайт должен обладать адаптивной под размер экрана версткой и мобильной версией; 
Концепция Desktop first; 
Верстку мобильной версии можно упростить (контуры на бэкграунде); 
Доступность и скорость:
Сайт должен быть доступным и работать для десктопа (веб) и мобильных браузеров: Google Chrome, Mozilla Firefox, Opera, Internet Explorer, Facebook Browser (нативный браузер при переходе на внешние ресурсы из Instagram, Facebook);
Сайт должен иметь показатель 90-100 по Google Speed Test:  https://developers.google.com/speed/pagespeed/insights/ 
Оптимизация под поиск (seo).
Передача кода: 
Разработчик должен передавать все необходимые для работы сайта и подключение к домену mirelia.co файлы и доступы.
